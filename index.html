Git Tutorial

Why git is important for developers? It makes it easier to colaborate with other and share other projects. It also allows to track your work anytime.. while you work you can sort of take a snapshot of your project so later on in the future you can reutilize or rebuild your work.

Useful Git commands

  • git status (tells you if you made any modifications)

When you start your own repository, you run the following commands:

  • git init (it informs the software that it needs to treat the current directory where you are working on as a repository). YOU ONLY RUN IT ONCE TO INITIALIZE THE DIRECTORY AS A REPOSITORY
  • git add -A (it tells git make sure that you keep an eye recognize the changes that I made to all files and directories inside the repository)
  • git commit -m "YOUR COMMIT MESSAGE" (this is what you do to create a remote backup of your code since your code before this commands was only living on your local machine)
  • git remote add origin URL_TO_REMOTE_REPO
  • git push origin main or git push origin master
